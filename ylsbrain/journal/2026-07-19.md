**What did NOT work (big lesson):** synthetic-pointer grabs POISON the turn

its label (React state ≠ hook state — label change is NOT proof the turn
ran). Cost ~30min of forensics; remount (tab toggle) resets. Real mice are
untouched. ALWAYS stub capture fns before synthetic sheet-pointer tests.

### [18:21] Pan made explicit (right-drag/2-finger), zoom-threshold auto-switch removed   (branch: feature/designer-3d-preview · commits: d198a87)

fix made left-drag switch orbit→pan only past an invisible camera-distance
threshold (1.15), so nothing signaled when pan was "on". Replaced with THREE
FIXED gestures, none conditional on state: left-drag=rotate always,
right-drag/two-finger=pan ALWAYS (any zoom), edge-band grab=turn page.

per-frame conditional swap; removed ZOOM_PAN_DIST entirely.

BOTH default zoom and deep zoom without needing to zoom in first, no flip
triggered, flip-button label unchanged; left-drag orbited (slight rotation

correct fix: discoverability > cleverness — an auto-detected mode nobody can
see beats nothing, but an always-on fixed gesture beats both.
